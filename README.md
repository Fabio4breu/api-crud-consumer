# Aplicativo Console – Consumo de API

Projeto em Node.js que consome uma API Express utilizando Axios
e entrada de dados com readline-sync.

## Como executar

### 1. Instalar dependências
npm install

### 2. Executar a API
node index2.js

### 3. Executar o cliente
node index.js
